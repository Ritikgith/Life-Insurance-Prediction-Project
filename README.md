# Life-Insurance-Prediction-Project
The project aimed to develop a predictive model to estimate the likelihood of policyholders making a life insurance claim within a specified period. The primary goal was to enhance the underwriting process, improve risk management, and optimize the pricing strategy for life insurance policies.
Technologies : Python , Machine Learning , Power BI , Pandas , Numpy , Seaborn ,       
                         MySQL, Scikit-lear
